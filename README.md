# PrimalTraining Web Application

Профессиональный проект для фитнес-клуба, реализованный на современном стеке технологий: React, TypeScript, Vite, Tailwind CSS. Проект предназначен для демонстрации и практики команд Git, а также для создания удобного и стильного веб-интерфейса для пользователей.

## Стек технологий

- **React** — компонентный подход, современный UI
- **TypeScript** — типизация, надежность кода
- **Vite** — быстрый сборщик и dev-сервер
- **Tailwind CSS** — гибкая и быстрая стилизация
- **React Router** — маршрутизация страниц
- **ESLint, Prettier** — поддержка качества и стиля кода

## Структура проекта

- `web/src/pages` — основные страницы: Home, About, Reserve
- `web/src/components` — переиспользуемые компоненты: NavBar, Footer, Card, Buttons
- `web/src/assets` — кастомные цвета и шрифты
- `web/public` — статические ресурсы (логотипы, изображения)
- `web/index.html` — точка входа

## Основные реализованные функции

- **Навигация**: адаптивное меню, поддержка мобильных устройств
- **Главная страница**: современный дизайн, секции Header, MainContent, ReserveHome
- **About**: подробная информация о клубе, миссия, команда
- **Reserve**: форма бронирования персональных тренировок
- **Карточки классов**: компонент `ClassCard` с динамическими данными и кнопкой записи
- **Кастомные кнопки**: PrimaryButton, SecondaryButton с поддержкой темизации
- **Tailwind CSS**: кастомные стили, шрифты, цвета
- **Маршрутизация**: переходы между страницами через React Router
- **Адаптивность**: поддержка всех устройств

## Как запустить

1. Установите зависимости:
   ```bash
   npm install
   cd web && npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
3. Откройте [http://localhost:5173](http://localhost:5173) в браузере
